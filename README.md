## Install

This project requires [**Node.js**][nodeversion-url] **v6** or above. Install it using [**yarn**](https://yarnpkg.com) **v1** or above / [**npm**](https://www.npmjs.com) **v5** or above.

```
$ yarn add rollup-plugin-just-gzip --dev
```
